Банковская система
Комплексная реализация банковской системы на C++ с использованием объектно-ориентированного программирования, полиморфизма и умных указателей.

Основной функционал
1. Управление клиентами: Создание и управление обычными и премиум-клиентами
2. Управление счетами: Поддержка расчетных и сберегательных счетов
3. Обработка транзакций: Вклады, снятия и переводы
4. Интерфейс с меню: Удобный консольный интерфейс
5. Обработка ошибок: Комплексная система исключений

Типы счетов
1. Расчетный счет: Возможность овердрафта с динамическими лимитами
2. Сберегательный счет: Начисление процентов с переменной ставкой

Расширенные возможности
1. Умные указатели для управления памятью
2. Полиморфный дизайн с виртуальными функциями
3. Поддержка системы сборки CMake
4. Комплексная система логирования
5. Отслеживание истории транзакций

Инструкция по сборке
- Компилятор с поддержкой C++17 (GCC 7+, Clang 5+, MSVC 2017+)
- CMake версии 3.15 или выше

Особенности реализации
—————Расчетный счет—————
Овердрафт: Динамический лимит в зависимости от баланса
Комиссия: 2% за каждые 500₽ снятия (макс. 20%)
Автоматическое увеличение лимита: При балансе >50,000₽

—————Сберегательный счет—————
Минимальный баланс: 5,000₽
Процентная ставка: Зависит от суммы на счете
Срок: Фиксированный срок в месяцах
